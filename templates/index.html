How it works

  1. Click “Choose .ipynb file” and select your notebook.
  2. Hit “Convert & Download”.
  3. You’ll get a ZIP with:
    • One .py per code cell, named like notebook_cell_001.py.
    • A combined.py that merges all code cells (in order).
